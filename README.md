# <h1>Youtube-clone</h1>

Youtube Clone is a React App that looks like the real Youtube. 

<ul><h3>The Stacks I've used are : </h3></ul>
  <li>React JS for the Front-End</li>
  <li>Node JS with Express JS for the Back-End</li>
  <li>Axios</li>
  <li>Material UI</li>

<h3>Installation</h3>
```sh
cd dillinger
npm i
npm start
```
